# Codex 异构分析请求：稳定币"低买高卖 + 理财"策略

## 背景与用户需求（老板的真实意图）

在 **Bybit 现货**做稳定币"低买高卖"，标的是 **0 手续费**的稳定币对。核心想法：稳定币在 1 附近波动，低于均值买、回归后高于买价卖，赚价差；持有期间还能吃 Bybit **统一交易账户(UTA)的持有利息**。用户贴的 `ArbiStrategy.py`（freqtrade）就是这个策略的实现，逻辑：
- 5m 执行 + 1h informative（EMA21/55/100）。
- 入场：`confirm_trade_entry` 仅当 `市价 - ema55_1h < 0.0001`（约1bp）才放行；`custom_entry_price` 限价买在 `min(市价, ema55_1h)`，下跌趋势再 −1bp。
- 出场：`custom_exit_price` 目标 = `买价 + 1bp`（ema55 上升趋势则 +2bp）；2 天后降档到 +1bp；`confirm_trade_exit` 规则：利润≥1.9bp→卖 / 2天内禁卖（为吃利息）/ 3天后即便≤买价也卖 / 否则亏不卖。
- `stoploss=-0.99`（实质无止损）、`position_adjustment_enable=False`（无 DCA）。

## 已确认的事实（来自实时 Bybit API + 用户）
- 0 手续费是**促销**（非永久），曾有 USD1 嘉年华 2026-04~05；用户确认当前这几个对 0 fee。
- **tickSize = 0.0001 = 1bp 是价差地板**；盘口价差常年贴地 1-2bp。
- **UTA 持有利息（用户提供，2026-06-14）：USD1 = 10%，USDe = 3.5%，USDtb = 3.5%；USDT/USDC 持有不计息。**
- 用户最新决策：**只交易 USD1**；持有 USD1 久 = OK（吃 10% 利息）；**尽量别让 USDT 空闲**（0 息）；**假设不发生真脱锚，USD1 必回锚**。
- USD1 6.7 个月 5m 数据：均值 0.9998，约 68% 时间 < 1.0，中位"低于均值"幅度约 −4bp，最深日线到过 0.994。

## 我（Claude）的分析结论（请 Codex 复核/挑错）

用自建事件驱动回测（忠实复刻上面 callback 逻辑，6.7 个月真实 5m 数据，单仓、无止损、利息按持有时长叠加），并对**逆向选择 adverse-selection（adv，每次被动成交比报价差几 bp）**做敏感性扫描：

USD1 完整策略（低买高卖 + 10% 利息）年化：
- adv=0（完美成交）: 11.07%（> 持有 10%）
- adv=0.5bp: 9.57%（< 持有）
- adv=1.0bp: 9.25%（< 持有）
纯价差部分（不含利息）：adv=0 时 2.1%，adv≥1bp 时 ≈0 或转负。
在 USDT 空闲时间约 7-10%，× 10% 利息 ≈ 0.7-1%，正好是"交易 < 持有"的缺口。

**我的核心结论：**
1. 这个策略的年化几乎完全由 adv 决定。adv<~0.4bp 时交易USD1(11%)略胜持有(10%)；adv≥0.5bp 时交易反而**跑输直接持有 USD1 的 10%**。
2. 纯"低买高卖"价差 edge 在现实成交下≈0（每笔毛利~1.8bp，adv 吃掉≈0.9bp/边就归零）。
3. "亏不卖"不能消除 adv：它把 adv 变成**资金被套住**（USD1 有仓位套 100+ 天）+ 机会成本，而非账面亏损 → 高胜率(86-97%)是假象。
4. freqtrade 那份代码还有 `has_open_trade/custom_info` 门控导致**回测失真**（回测里 enter 恒开/exit 信号恒不触发，回测≠实盘）—— 所以我没用它回测，改自建。
5. tickSize=1bp 是价差地板；5m OHLC 无法测 adv，需 tick 级 dry-run 实测（已建工具 dryrun.py）。
